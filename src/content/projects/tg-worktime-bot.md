## Зачем

Оплата на работе считается по часам, проведённым на месте. Нужен был быстрый способ
отмечать приход и уход и видеть предполагаемый заработок за месяц - без ручного счёта.
Сделал бота под себя, пользуюсь каждый день.

## Что умеет

- Отметка прихода и ухода - кнопками в меню или командами (`/in`, `/out`), можно указать
  конкретное время вручную
- Запись в личную Google-таблицу учёта через сервисный аккаунт
- Месячный отчёт - приходит картинкой, с выбором месяца из последних 12

## Решение, которым доволен

Бот пишет в таблицу только две вещи - время прихода и время ухода. Всё остальное (часы за
день, заработок, итоги месяца) считают формулы в самой Google-таблице. Это сознательное
разделение: бот отвечает за ввод данных, таблица - за расчёты. Логику подсчёта можно менять
прямо в таблице, не трогая код бота.

## Технически

- `python-telegram-bot` для интерфейса
- `gspread` + сервисный аккаунт Google для работы с таблицей
- `matplotlib` для генерации отчёта-картинки
- Развёрнут на VPS, работает постоянно
- Доступ ограничен одним пользователем (проверка по Telegram user id)

## Что показывает

Маленький, но завершённый и реально работающий инструмент: внешний API (Google Sheets),
генерация изображений, аккуратная работа с секретами (`.env`, сервисный аккаунт, ничего
лишнего в репозитории), деплой на постоянно работающий сервер.
