## Задача

Ранее колебания испытуемых объектов измерялись акселерометрами - физическими датчиками,
которые нужно крепить непосредственно на объект. Задача - реализовать бесконтактный способ
измерения: следить за маркерами на объекте по видеопотоку и в реальном времени вычислять и записывать в CSV файл
амплитуду и частоту колебаний каждого из них.

## Что сделал

Изначально проект был приложением для работы с камерами. Я добавил в него поиск AruCo-маркеров
и расчёт их параметров колебаний, после чего всё это итеративно дорабатывалось.

- Реализовал поиск и отслеживание AruCo-маркеров с получением пиксельных и физических координат
- Реализовал расчёт амплитуды и частоты колебаний каждого маркера по временному ряду координат
- Достиг **80 измерений в секунду на каждый маркер** - в реальных испытаниях одновременно
  отслеживалось до 7 маркеров на одном объекте
- Реализовал визуализацию в реальном времени: графики координат X/Y, смещений от старта,
  частоты и амплитуды для каждого маркера
- Реализовал перевод пиксели → миллиметры через масштабный коэффициент по заданным
  физическим размерам марки, с переключением единиц измерения прямо в UI
- Реализовал взаимодействие с одноплатным компьютером по UDP - он выступал внешним триггером
  и обеспечивал захват кадров с частотой 80 FPS
  (серверная часть на одноплатнике написана коллегой)

## Преимущество перед акселерометрами

Бесконтактный метод позволяет следить за несколькими точками объекта одновременно с одной
камеры - без монтажа датчиков на каждую точку.
